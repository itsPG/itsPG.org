About

    I'm PG, a.k.a. itsPG. My name is Tsung-Han Tsai (蔡宗翰). I completed senior high school in TNFSH. In 2006, I entered NSYSU and graduated in 2011 with a bachelor's degree in CSE. During my college years, I practiced my programming skills and my work is recongnized by getting good scores in programming contests. In 2011, I chose to pursue my master's degree in the Network Engineering College of Computer Science of NCTU and I finished my oral defense in Nov, 2013.

    I enjoy coding, especially when I'm writing something amazing that does improve others' life. I'm skilled in C/C++ since I've used C++ to solve over 500 problems in UVa Online Judge. I also use JavaScript and Python to help me finish my tasks. I manage my codes by git, pay attention to unit tests, and think KISS (Keeps It Simple and Stupid) is the best principle in programming.

Portfolio

Anime Cloud

It indexes the .mp4 files in hard-disk automatically and provides a list through httpd server.

此程式能夠掃描硬碟內的符合指定名稱的mp4檔並加以整理，以及用httpd server的形式分享整理後的清單。

Torrent hunter

It parses a website page, compares the content, and download the torrents that match our requests.

此程式能夠讀取以及分析網頁，將內容與目標清單做比較，並根據目前的需求下載＆讀入指定的種子檔。

99manga preloader

It speeds up the website by preloading the images. You may download it from chrome web store.

此程式利用預讀的方式加速99manga網站。 本擴充套件可於chrome應用程式中取得。

Maxflow Mincost

I implemented maxflow mincost algorithm. It solves flow problems and draw the graph by animation.

我用JS實做出maxflow mincost演算法，可以拿來解分配問題以及網路流問題，同時會用動畫呈現計算過程。

RTP player and server

I implemented simplified RTP/RTSP protocol and built buffer to sort the out of order UDP packets.

我實做了簡化版的RTP/RTSP即時串流協議，並利用buffer可以解決潛在的封包順序錯亂的問題。

Mikochan

It's an image-based bulletin board system which supports thumbs, threads, and comments. I spent lots of time dealing with the UI.

此程式為貼圖取向的留言系統，支援自動縮圖，討論串，訪客留言等功能。我花了相當的時間在處理UI部份，希望能達到簡潔易用的目標。

ColorPlus.js

It provides a painless way to color the console in node.js by outputting ANSI escape codes.

此程式透過輸出ANSI色碼的方式幫node.js的console上色。

JavaScript Parser

It parses JavaScript and builds ASTs in JSON format.

此程式可以解析JavaScript，並用JSON的形式輸出parse後的AST。

Diff.js

I implemented diff module for node.js. It use LCS algorithm to show the difference between two strings.

我在node.js中用JavaScript實做出diff模組，採用LCS演算法比較出兩字串的相異之處並顯示。
